# TextFormation

TextFormation is simple rule system that can be used to implement typing completions. Think matching "}" with "{".

## Integration

Swift Package Manager:

```swift
dependencies: [
    .package(url: "https://github.com/ChimeHQ/TextFormation")
]
```
